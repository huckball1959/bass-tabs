Huckball BassTab Studio Pro - Help

1) Core Workflow

  • Select a mode with the mode button: CREATE, PLAYBACK, or AUDITION.
  • In CREATE mode, click staff lines to place notes and click chord lane to place chord symbols.
  • Use PLAY to hear playback with optional metronome/chord/organ-sustain features.
  • Use Save/Load to store songs as .bass files.

2) Modes

  • CREATE: Add and edit notes/chords.
  • PLAYBACK: Click staff to place playhead without creating notes.
  • AUDITION: Click notes/strings/chords to hear sound quickly.
  • Shortcut: Alt+T cycles modes.

3) Measure Selection and Clipboard

  • Click a barline to select one measure; Ctrl+click to multi-select/toggle.
  • Right-click selected measure(s) for copy options: all / notes only / lyrics only / chords only.
  • After copy, select destination measure and use Paste at this measure (auto-overwrite copied types).
  • Ctrl+C / Ctrl+V also work with measure clipboard flow.

4) Time Signatures

  • Global time signature: click TIME SIG near top and enter values like 4/4, 6/8, 5/7.
  • Per-measure override: select measure(s), then use Measure TS (or Alt+M).
  • Measure overrides display beside measure numbers in parentheses.

5) Sections (Verse / Chorus / Bridge)

  • Use the Sections button to define a section from selected contiguous measures.
  • Copy section by name/number, then paste to a destination measure.
  • Sections are saved with the song.

6) Playback Options

  • Chords toggle: include/exclude chord playback during transport.
  • Organ Sustain: hold chord until next chord (or stop/rewind).
  • Metronome toggle and BPM controls are available in top bar.

7) Editing Notes and Chords

  • Select note and type fret numbers, + or - to change fret.
  • Delete/Backspace removes selected notes.
  • Right-click note for note options; right-click chord for chord options.
  • Drag selected notes/chords to reposition.

8) Print and Export

  • Print view hides UI controls and playhead, and shows custom print header.
  • Save supports system Save As where available, with download fallback.
  • Load opens .bass song files into a new tab.

9) Keyboard Shortcuts (Main)

  • Space: Play/Stop
  • R: Rewind
  • Alt+T: Cycle modes
  • Alt+M: Set selected measure time signature
  • Alt+Z / Alt+Y: Undo / Redo
  • Ctrl+C / Ctrl+V: Copy / Paste
  • Esc: close menus/help
